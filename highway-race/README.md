# Highway Race

A fast-paced racing game where you race down the highway. Navigate through traffic and reach the finish line.

## Features

- Highway racing gameplay
- Traffic avoidance
- Speed mechanics
- Canvas-based graphics

## Local Development

```bash
npm install
npm run dev
```

The game will be available at `http://localhost:3000`

## Deployment

Deploy to Vercel:

```bash
vercel
```
