# Marble Labyrinth

Navigate a marble through a tilted maze. Tilt your screen with your mouse to roll the marble towards the goal while avoiding holes.

## Features

- Physics-based marble movement
- Tilt/mouse control
- Maze obstacles and holes
- Score tracking

## Local Development

```bash
npm install
npm run dev
```

The game will be available at `http://localhost:3000`

## Deployment

Deploy to Vercel:

```bash
vercel
```
