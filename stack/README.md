# Stack

A stacking block puzzle game. Stack blocks accurately to build higher and higher!

## Features

- Block stacking mechanics
- Alternating horizontal and vertical placement
- Progressive difficulty
- CSS-based animations

## Local Development

```bash
npm install
npm run dev
```

The game will be available at `http://localhost:3000`

## Deployment

Deploy to Vercel:

```bash
vercel
```
