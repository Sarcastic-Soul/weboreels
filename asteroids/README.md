# Asteroids Game

A classic asteroids arcade game built with vanilla JavaScript and Canvas API.

## Features

- Classic asteroids gameplay
- Weapon system with special items
- Score tracking
- HUD display

## Local Development

Install dependencies and start development server:

```bash
npm install
npm run dev
```

The game will be available at `http://localhost:3000`

## Deployment

Deploy to Vercel:

```bash
vercel
```

## Requirements

- Node.js 14+ for local development
- Modern web browser with Canvas API support
