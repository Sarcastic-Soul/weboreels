# Super Hopper

A cute 3D platformer adventure game built with Three.js. Jump and hop your way through levels while avoiding obstacles.

## Features

- 3D graphics using Three.js
- Platformer gameplay with jumping mechanics
- Multiple levels
- Score tracking
- Keyboard controls (Arrow keys and Up arrow for jump)

## Local Development

```bash
npm install
npm run dev
```

The game will be available at `http://localhost:3000`

## Deployment

Deploy to Vercel:

```bash
vercel
```

## Requirements

- Modern web browser with WebGL support
- Three.js is loaded from CDN via import maps
