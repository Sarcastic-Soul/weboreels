# Coloron

A color-matching game where you match bouncing balls with colored bars.

## Features

- Dynamic ball bouncing with color changes
- Tap to switch bar colors
- Color matching gameplay
- Smooth animations

## Local Development

```bash
npm install
npm run dev
```

The game will be available at `http://localhost:3000`

## Deployment

Deploy to Vercel:

```bash
vercel
```
