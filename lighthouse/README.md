# Lighthouse

A lighthouse-themed interactive experience.

## Features

- Lighthouse theme
- SCSS styling

## Local Development

```bash
npm install
npm run dev
```

The game will be available at `http://localhost:3000`

## Building

To compile SCSS:

```bash
npm run build
```

## Deployment

Deploy to Vercel:

```bash
vercel
```
